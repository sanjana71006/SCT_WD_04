# Smart To-Do List Web App

## Project Description

TaskNest is a modern, responsive to-do list application that helps you organize your tasks efficiently. With a clean interface and powerful features, TaskNest makes task management simple and enjoyable.

Key Features:
- 📝 Create, edit, and delete tasks with ease
- 🏷️ Categorize tasks (Work, Personal, Shopping, etc.)
- 🔼 Prioritize tasks (High, Medium, Low)
- ⏰ Set due dates and times with reminders
- 🌙 Dark/Light theme toggle
- 📱 Fully responsive design for all devices
- 📲 Progressive Web App (PWA) support
- 🔔 Desktop notifications for due tasks

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript
- **Icons**: Font Awesome
- **Font**: Poppins (Google Fonts)
- **Storage**: LocalStorage
- **PWA**: Service Worker, Web App Manifest

## Installation

To run TaskNest locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/tasknest.git
   cd tasknest
   ```

2. Open `index.html` in your browser (no server required as it runs client-side)

## Usage

1. **Add a Task**: Click the + button in the bottom right corner
2. **Edit a Task**: Click the edit icon on any task
3. **Complete a Task**: Check the checkbox next to a task
4. **Delete a Task**: Click the trash icon on any task
5. **Filter Tasks**: Use the category and priority filters at the top
6. **Toggle Theme**: Click the moon/sun icon in the header
